# First Kirill Game Web

Открывай сайт из папки `mobile`.

Главный файл:

- `index.html`

Чтобы открыть на телефоне:

1. Залей папку `mobile` на любой хостинг или GitHub Pages.
2. Открой ссылку в Safari на iPhone или в Chrome на Android.
3. При желании добавь сайт на главный экран.

Что уже есть:

- заставка `FIRST KIRILL GAME` примерно на 6.5 секунд
- игра змейка
- управление свайпами и кнопками
- сохранение рекорда
